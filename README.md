# BookCourier Server

The robust backend API for the BookCourier e-commerce platform. Built with Node.js, Express, and MongoDB, securely managed with Firebase Authentication.

## Features

## Tech Stack

- **Runtime**: [Node.js](https://nodejs.org/)
- **Framework**: [Express.js](https://expressjs.com/)
- **Database**: [MongoDB](https://www.mongodb.com/) (Mongoose ODM)
- **Authentication**: [Firebase Admin SDK](https://firebase.google.com/docs/admin/setup)
- **Deployment**: [Vercel](https://vercel.com/)
